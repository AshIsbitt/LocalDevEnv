### How to Run

1) To pull from Github:
```LOREM IPSUM```

2) To run this vagrant installation:
```vagrant up```

3) To enter into the virtual machine:
```vagrant ssh```

4) To run the app:
```
cd /home/ubuntu/app
npm install express
npm install mongoose
npm install ejs
npm start
```
